MEMORY GAME

HOW TO PLAY

Upon pressing play, a board of 12 cards will appear and one card will light up.
You must select that card in order to gain a point.
The computer will then add a card to it's sequence and light them both up in the order you have to select them.
The number of points you have corresponds to the length of the sequence.
You must guess the entire sequence before the game will tell you if you're right or wrong.
